FAQ

Common questions

Here are some of the most common questions asked about Terra.
What is Terra? Terra is a Discord bot that allows you to learn more about the world through play. Start a game of Guess, Worldle, Factle, and more to find out more about a given place.
How can I play Guess on Terra? To play Guess on Terra, simply invite the bot to your Discord server and type '/play' to start a game.
Can I customize my user experience on Terra? Of course! Terra allows you to set your own country and units to personalize you user experience. The bot will also support languages other than English soon.
Is Terra suitable for beginners? Absolutely! Terra provides fun, interactive games that can be enjoyed by anyone, whether pro or beginner.
How can I report a bug or suggest a feature for Terra? If you encounter any bugs or have suggestions for new features, please join our support server and let us know. We appreciate your feedback!
